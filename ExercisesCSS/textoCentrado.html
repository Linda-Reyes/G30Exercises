CSS_ Hojas de Estilo de Cascada

CSS (Cascading Style Sheets) es un lenguaje que describe la presentación y el diseño de documentos HTML o XML, permitiendo controlar la apariencia visual de las páginas web. Utiliza selectores para aplicar estilos específicos a elementos, definiendo propiedades como color, fuente, tamaño, márgenes, y más, y sigue principios de cascada y especificidad para determinar cómo se aplican estos estilos.
