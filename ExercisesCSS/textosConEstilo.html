HTML - CSS - JavaScript

HTML (HyperText Markup Language)

HTML es el lenguaje estándar de marcado utilizado para crear y estructurar el contenido en la web. Define la estructura básica de una página web mediante el uso de elementos y etiquetas, que permiten incluir texto, imágenes, enlaces y otros elementos multimedia. HTML proporciona el esqueleto de una página web, organizando el contenido en una jerarquía lógica que puede ser interpretada por los navegadores web para mostrar la información de manera coherente y accesible.

CSS (Cascading Style Sheets)

CSS es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS controla el aspecto visual de las páginas web, incluyendo el diseño, los colores, las fuentes y el espacio entre los elementos. Al separar el contenido (HTML) de la presentación (CSS), los desarrolladores pueden mantener y actualizar el diseño de manera más eficiente, garantizando una apariencia consistente y atractiva en diferentes dispositivos y navegadores.

JavaScrip

JavaScript es un lenguaje de programación de alto nivel y dinámica que se utiliza para crear y controlar contenido interactivo en páginas web. A diferencia de HTML y CSS, que son lenguajes de marcado y estilo respectivamente, JavaScript permite añadir funcionalidad a las páginas web, como validaciones de formularios, animaciones, y la manipulación de datos en tiempo real. Es una parte esencial del desarrollo web moderno y, junto con HTML y CSS, constituye la base de la mayoría de las aplicaciones web interactivas.
